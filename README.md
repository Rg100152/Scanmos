## 📌 Overview

**ScanMos** is a professional-looking, animated CLI network scanner simulator built entirely with Python's standard library – **zero external dependencies**. It demonstrates advanced terminal UI techniques, animations, and report generation, making it perfect for learning, portfolio, and mobile use.

> ⚠️ **This is an educational simulation tool. It does NOT perform real network scanning.** All results are randomly generated for demonstration purposes.

---

## ✨ Features

### 🎨 Professional Hacker-Style UI
- Animated ASCII logo with vibrant colors
- Typewriter effects, loading bars, and spinner animations
- Pulsing text and smooth transitions
- Colour-coded output (works on all terminals)

### 🔍 5 Scan Modes (Simulated)
| Mode | Description |
|------|-------------|
| **Quick Scan** | Fast, minimal port check |
| **Fast Scan** | Scans common ports only |
| **Deep Scan** | Thorough port and service detection |
| **Stealth Scan** | Simulated stealth technique |
| **Full Scan** | Complete comprehensive scan |

### 📊 Realistic Simulated Results
- Host discovery (IP, MAC, vendor, OS)
- Open ports with service names
- Service banners (fake but realistic)
- Security risk rating with score (Low/Medium/High)
- Latency, TTL, uptime, and traffic stats

### 📁 Export Reports
- **JSON** – Machine‑readable format
- **TXT** – Plain text report
- **HTML** – Professional web report with styling

### 📋 Scan History
- Stores all scan results in memory
- View past scans with detailed information

### 💻 System Info
- Display OS, architecture, Python version, hostname, and processor

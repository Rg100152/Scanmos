## 📌 Overview

**ScanMos** is an educational Python CLI application inspired by high-speed network scanners. It simulates a professional penetration testing interface with animated terminal output, colorful dashboards, progress indicators, and realistic-looking scan results.

> **⚠️ Note:** This project is for learning and demonstration only. It does **not** perform real network scanning.

---

## ✨ Features

### 🎨 User Interface
- Animated ASCII logo with `pyfiglet`
- Rich color palette (Purple, Blue, Cyan, Green, Yellow, Red)
- Smooth startup loading animation
- Hacker-style terminal panels

### 🔍 Scan Modes (Simulated)
- Quick Scan
- Fast Scan
- Deep Scan
- Stealth Scan
- Full Scan

### 📊 Simulated Results
- Host discovery (IP, MAC, vendor, OS)
- Open ports with service names
- Banner grabbing (fake banners)
- Security risk rating (Low/Medium/High)
- Latency and TTL information

### 📁 Reporting
- Export to **JSON**, **TXT**, **HTML**
- Scan history tracking
